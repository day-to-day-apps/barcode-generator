Waarom Code 128?

Snelle generatie met automatische validatie.

  • Ondersteunt de volledige ASCII-set 0–127.
  • Automatische keuze van subset A, B of C voor de beste dichtheid.
  • Mod-103-checksum wordt automatisch toegevoegd.
  • Geweldig voor logistiek, verzendetiketten en magazijntracering.

Gebruiksscenario's voor Code 128

Perfect voor het printen van labels, pakbonnen en productlabels.

  • Verzend- en koerieretiketten voor UPS, FedEx en DHL.
  • Magazijn- en voorraadtracering.
  • Product-IDs, verpakkingen en palletetiketten.
  • GS1-128-barcodes met toepassingskenmerken.

Veelgestelde vragen

Welke tekens ondersteunt Code 128?

De meeste Code 128-standaarden ondersteunen alfanumerieke tekens en een paar symbolen. De generator controleert automatisch het juiste tekenreeks.

Is Code 128 geschikt voor verzendetiketten?

Ja. Code 128 wordt veel gebruikt in verzending, logistiek en magazijn wanneer een compacte en betrouwbare barcode nodig is.

Hoe exporteer ik Code 128 naar PNG of SVG?

Gebruik de exportopties op de pagina om je code te downloaden als PNG of SVG. De code is printklaar.

Kunnen scanners Code 128 lezen?

Ja. Code 128 wordt ondersteund door de meeste 1D-scanners, mobiele scanapps en industriële lezers.
