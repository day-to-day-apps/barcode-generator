Obsługiwane standardy kodów kreskowych

Code 128

Uniwersalny kod obsługujący pełny zestaw znaków ASCII. Idealny do logistyki, etykiet i identyfikacji.

EAN-13 / EAN-8

Europejski standard dla produktów handlowych. EAN-13 to 13 cyfr, EAN-8 to wersja skrócona.

UPC-A

Amerykański standard kodów produktowych. 12-cyfrowy kod stosowany głównie w Ameryce Północnej.

Code 39

Alfanumeryczny kod obsługujący wielkie litery, cyfry i kilka znaków specjalnych. Popularny w przemyśle.

ITF / ITF-14

Interleaved 2 of 5 — kod numeryczny stosowany na opakowaniach zbiorczych i w logistyce.

Codabar / Pharmacode

Codabar — biblioteki i banki krwi. Pharmacode — branża farmaceutyczna do kodowania opakowań leków.

Jak wygenerować kod kreskowy

  1. Wybierz format

    Wybierz standard kodu z listy — EAN-13, UPC-A, Code 128, QR Code, ITF i inne.

  2. Wpisz dane

    Wpisz liczbę lub tekst do zakodowania. Narzędzie na bieżąco sprawdza sumy kontrolne i długość.

  3. Dostosuj wygląd

    Zmień rozmiar, kolory, marginesy i obrót. Podgląd aktualizuje się natychmiast.

  4. Pobierz lub drukuj

    Eksportuj jako PNG lub SVG, albo otwórz okno druku etykiet dla drukarek termicznych i arkuszy A4.

Najczęściej zadawane pytania

Czy generator kodów kreskowych jest naprawdę darmowy?

Tak — w pełni darmowy, bez rejestracji, bez znaków wodnych i bez limitu liczby generowanych kodów.

Czy moje dane są prywatne?

Tak. Generowanie odbywa się w 100% w przeglądarce. Nic nie jest wysyłane na żaden serwer — dane zostają na Twoim urządzeniu.

Jakie formaty kodów są obsługiwane?

18 standardów, w tym EAN-13, EAN-8, UPC-A, UPC-E, Code 128, Code 39, ITF-14, Codabar, MSI, Pharmacode i QR Code.

Czy mogę zdekodować kod kreskowy ze zdjęcia?

Tak — użyj Dekodera kodów kreskowych. Wgraj zdjęcie lub wklej zrzut ekranu i odczytaj EAN, UPC, Code 128, QR i inne.

Czy można drukować etykiety?

Tak — narzędzie zawiera okno druku etykiet dla drukarek termicznych (Zebra, Brother, Dymo) oraz arkuszy A4 (układy Avery).

Czy mogę używać kodów komercyjnie?

Tak. Kody możesz używać w dowolnym projekcie komercyjnym i prywatnym. Numery EAN/UPC do sprzedaży detalicznej nadal muszą być zarejestrowane w GS1.
