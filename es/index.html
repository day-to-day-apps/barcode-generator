Estándares de códigos de barras compatibles

Code 128

Código de barras universal compatible con el conjunto completo de caracteres ASCII. Ideal para logística, etiquetas e identificación.

EAN-13 / EAN-8

Estándar europeo para productos de venta al por menor. EAN-13 usa 13 dígitos, EAN-8 es la versión corta.

UPC-A

Estándar americano de código de producto. Un código de barras de 12 dígitos utilizado principalmente en América del Norte.

Code 39

Código de barras alfanumérico que admite letras mayúsculas, dígitos y algunos caracteres especiales. Popular en la fabricación.

ITF / ITF-14

Interleaved 2 of 5 — un código de barras numérico utilizado en cajas de envío y logística.

Codabar / Pharmacode

Codabar — bibliotecas y bancos de sangre. Pharmacode — industria farmacéutica para la codificación de envases de medicamentos.
