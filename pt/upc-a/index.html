Perguntas frequentes

Quais caracteres UPC-A suporta?

A maioria dos padrões UPC-A suporta caracteres alfanuméricos e alguns símbolos. O gerador valida automaticamente o conjunto correto.

UPC-A é adequado para etiquetas de envio?

Sim. UPC-A é muito usado em remessas, logística e armazém quando se precisa de um código compacto e confiável.

Como exporto UPC-A para PNG ou SVG?

Use os controles de exportação na página para baixar seu código como PNG ou SVG. O código fica pronto para impressão.
