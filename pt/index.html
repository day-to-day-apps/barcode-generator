Padrões de códigos de barras suportados

Code 128

Código de barras universal que suporta o conjunto completo de caracteres ASCII. Ideal para logística, etiquetas e identificação.

EAN-13 / EAN-8

Padrão europeu para produtos de retalho. O EAN-13 utiliza 13 dígitos, o EAN-8 é a versão mais curta.

UPC-A

Padrão americano de código de produto. Um código de barras de 12 dígitos usado principalmente na América do Norte.

Code 39

Código de barras alfanumérico que suporta letras maiúsculas, dígitos e alguns caracteres especiais. Popular na indústria.

ITF / ITF-14

Intercalado 2 de 5 — um código de barras numérico usado em caixas de expedição e na logística.

Codabar / Pharmacode

Codabar — bibliotecas e bancos de sangue. Pharmacode — indústria farmacêutica para codificação de embalagens de medicamentos.
