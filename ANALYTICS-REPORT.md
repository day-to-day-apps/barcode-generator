# GA4 + Search Console report

Create a GA4 Exploration named **Organic tool funnel** with a weekly date range:

1. `session_start` filtered to `google / organic`.
2. `bulk_csv_import` or `generate_barcode`.
3. `bulk_export_pdf`, `bulk_export_zip`, `save_barcode` or `print_job_saved`.
4. `sign_up`.

Break down by landing page, country, device category and the `code_type` or `preset` event parameter. Add Search Console queries and landing pages beside the funnel. Review organic entrances, tool-start rate, successful-export rate, registration rate and export errors every week.
